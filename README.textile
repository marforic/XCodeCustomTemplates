<h2>INSTALLATION</h2>

* To install the custom Templates run:
 
<pre>
git clone git@github.com:marforic/XCodeCustomTemplates.git
cd XCodeCustomTemplates/
chmod u+x ./install_templates.sh
./install_templates.sh "Name Surname" "Email" "Company name"
</pre>

Please put double quotes as shown in example!
